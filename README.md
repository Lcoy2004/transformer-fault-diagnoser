## 项目简介

本项目是一款变压器故障诊断软件，采用**油色谱分析（DGA）**和**局部放电超声波检测（AE-PD）**监测技术，结合**PCA降维**和**随机森林分类/回归**算法，实现对变压器故障类型和定位的智能诊断。

### 监测技术说明

| 技术 | 传感器类型 | 测量物理量 | 诊断能力 |
|------|-----------|-----------|---------|
| **DGA（油色谱分析）** | 气体传感器 | 溶解气体含量（H₂、CH₄、C₂H₆、C₂H₄、C₂H₂） | 判断故障大类（正常/过热/放电） |
| **AE-PD（超声波局部放电检测）** | 超声波传感器（压电式） | 机械振动信号（电压输出） | 细化放电类型 + 空间定位 |

**关键区别**：本项目使用**超声波传感器**检测局部放电产生的机械振动信号，而非脉冲电流法检测电气信号。四通道超声波传感器布置于变压器不同位置，通过**到达时间差（TDOA）**算法实现放电点空间定位。

### 核心特性

| 特性 | 说明 |
|------|------|
| 多源数据融合 | 支持DGA油色谱数据 + 四通道超声波PD数据联合诊断 |
| 智能算法 | PCA降维 + 随机森林分类（故障类型）+ 回归（空间定位） |
| 三种诊断模式 | 仅DGA / 仅PD / 融合诊断 |
| 空间定位 | 四通道TDOA时差定位，输出三维坐标或时间差 |
| 可视化分析 | PCA贡献率图、样本分布图、诊断结果展示 |
| 数据管理 | SQLite数据库 + Excel数据导入 |

## 主要功能

1. **数据导入** - 导入DGA油色谱数据、四通道超声波PD数据（Excel格式）
2. **PCA降维** - 对四通道PD数据进行主成分分析
3. **模型训练** - 训练随机森林故障分类模型（类型）和回归模型（定位）
4. **智能诊断** - 三种模式：仅DGA、仅PD、融合诊断
5. **可视化** - 原始数据PCA贡献率柱状图、样本分布散点图、诊断结果展示
6. **日志管理** - 完整的操作日志记录

## 超声波PD特征说明

四通道超声波传感器（CH1~CH4）各提取12个特征，共48维特征。

### 特征类别与单位

| 特征类别 | 特征名称 | 单位 | 物理意义 |
|---------|---------|------|---------|
| **时域统计** | MEAN（均值） | mV | 信号平均电压 |
| | PEAK（峰值） | mV | 信号峰值电压 |
| | VAR（方差） | mV² | 电压波动程度 |
| **频域特征** | MAIN_AMP（主频幅值） | mV | 主频率分量幅值 |
| | MAIN_FREQ（主频率） | kHz | 信号主频率 |
| **频带能量** | BAND1~4_ENERGY | 无量纲 | 各频段能量占比（归一化0~1） |
| **形态统计** | KURTOSIS（峭度） | 无量纲 | 脉冲尖锐程度 |
| | SKEWNESS（偏度） | 无量纲 | 分布不对称程度 |
| **时间特征** | PULSE_WIDTH（脉冲宽度） | μs | 脉冲持续时间 |

### 数据格式示例

**DGA数据列**：H2, CH4, C2H6, C2H4, C2H2, fault_type, fault_location

**PD数据列**：ch1_band1_energy, ch1_band2_energy, ..., ch1_var, ch2_band1_energy, ..., ch4_var, fault_type, fault_location
（共48个特征列 + 2个标签列）

**fault_location说明**：
- 若提供传感器坐标和波速：三维空间坐标 `(x, y, z)`，单位米
- 默认情况：通道间到达时间差 `(Δt₁₂, Δt₁₃, Δt₁₄)`，单位秒

## 诊断模式

| 模式 | 说明 | 故障类型来源 | 定位信息来源 | 适用场景 |
|------|------|-------------|-------------|----------|
| **仅DGA** | 仅使用油色谱数据 | DGA模型 | DGA模型（若有） | 只有油色谱数据 |
| **仅PD** | 仅使用四通道超声波PD数据 | PD融合模型 | PD融合模型（TDOA时差定位） | 只有PD数据 |
| **融合诊断** | DGA判断大类 + PD细化放电类型 | DGA判断大类，PD细化类型 | **PD模型**（DGA无空间定位能力） | 两种数据都有 |

### 融合诊断决策逻辑

```
if DGA预测 == "放电":
    故障类型 = PD模型预测的放电细类（尖端/悬浮/沿面/气隙）
    故障定位 = PD模型的TDOA定位结果
else:
    故障类型 = DGA预测结果（正常/过热）
    故障定位 = DGA模型输出（若有）
```

**重要说明**：
- **DGA（油色谱分析）**：通过分析油中溶解气体判断故障类型，**不具备空间定位能力**
- **超声波PD**：四通道传感器通过信号到达时间差（TDOA）计算放电点位置，**具备空间定位能力**
- **融合诊断时**：若DGA判断为放电，定位信息必须取自PD模型，而非DGA模型

## 使用流程

```
数据导入 → PCA降维（PD数据） → 模型训练 → 智能诊断 → 查看结果
```

1. **导入数据** - 点击「导入数据」导入Excel数据（DGA数据或四通道PD数据）
2. **PCA降维** - 对四通道PD数据进行降维处理（提取主成分特征）
3. **训练模型** - 点击「训练判断模型」训练随机森林分类器（故障类型）和回归器（空间定位）
4. **智能诊断** - 输入待诊断数据（DGA/PD_CH1~4），点击「智能判断」
5. **查看结果** - 查看故障类型、定位坐标/时间差、置信度

## 软件界面

### 主界面功能区

- **数据表格** - 显示数据库中的训练/诊断数据

- **PCA贡献率图** - 显示各主成分的方差贡献率
- **样本分布图** - 显示二维PCA空间的样本分布
- **诊断结果** - 显示故障类型、定位结果（坐标或时间差）、置信度
